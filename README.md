# Publisher Rocket Reader

A macOS desktop app built with [Wails v2](https://wails.io) (Go + WebKit) that automates Publisher Rocket via Chrome DevTools Protocol (CDP) to extract category and keyword research data.

## What it does

- **Category Analyzer** — Navigates Publisher Rocket's Category Search UI, scrapes sales-to-#1, sales-to-#10, publisher %, KU %, insights, and keywords for each category path you provide. Outputs a markdown report.
- **CSV Analyzer** — Accepts a Competition Analyzer CSV exported from Publisher Rocket and returns an AI-generated markdown analysis. *(AI client TODO)*

## How it works

Publisher Rocket is an Electron app. This app launches it with `--remote-debugging-port=9222`, then connects via a raw WebSocket (Chrome DevTools Protocol). The Go backend injects JavaScript into Rocket's renderer to navigate the UI and scrape data — the same approach as the Author Tools VS Code extension, rewritten in Go.

## Prerequisites

- Go 1.21+
- Node.js 18+ / npm
- Wails CLI v2: `go install github.com/wailsapp/wails/v2/cmd/wails@latest`
- Publisher Rocket installed at `/Applications/Publisher Rocket.app`

## Getting started

```bash
# Install Go dependencies
go mod tidy

# Install frontend dependencies
cd frontend && npm install && cd ..

# Run in dev mode (hot reload)
wails dev

# Build production .app
wails build
```

The built app lands at `build/bin/Publisher Rocket Reader.app`.
